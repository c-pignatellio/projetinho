EcoStuff
A EcoStuff é um projeto desenvolvido por mim e mais três colegas durante dois anos, e integra um site, aplicativo de celular, aplicativo desktop e uma estufa controlada por Esp-32, com uma API desevolvida em Node.jspara controlá-la. Em sua maioria, o projeto foi dividido em tarefas para cada um dos membros, e mesmo que o projeto tenha sido terminado e apresentado, seu acesso não é amplo, e, com toda a honestidade, não há previsões de manutenção nem de disponibilização. Web - desenvolvido (por mim) em HTML, CSS e JavaScript Infelizmente, até a página do GitHub do site do projeto é privado, mas seguem abaixo imagens do site da EcoStuff :). O site tem a função de conectar o usuário à "empresa", ter seu primeiro contato, criar uma conta e eventualmente, até, realizar a compra do produto.
